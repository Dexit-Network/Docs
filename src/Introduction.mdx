# Introduction

The DeXit Network is high- performance transactions, fully compatible with EVM and support 
smart contract functionality today most of the Blockchain do not incentivize smart contract 
developers for their contribution to the network. 


DeXit Network intended to incentive for Smart contract Developer by offering up to 50% gas 
Fee on protocol level.  DeXit is decentralized, high-efficiency and energy-saving public chain. 
The Native  token of DeXit is DXT and it adopts the POSA consensus mechanism. 

The DeXit Network is client based on the go-ethereum and it offers an open-source framework 
for smart contracts. The project is an Ethereum sidechain that uses POSA consensus mechanism, 
validators are selected based on your staking weight to secure the network.
 The validators' identity is public.


The DeXit Network will allow Developer to build their Application and earn upto 50% transition fees
